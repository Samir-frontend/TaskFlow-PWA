# 📌 Overview

**TaskFlow PWA** is a modern and fully responsive **Task Management Progressive Web Application** built using **React, Vite and JavaScript**.

It helps users efficiently manage daily tasks with advanced filtering, smart statistics, notifications, calendar view, import/export functionality and installable PWA support.

Designed with a premium UI inspired by modern productivity applications.
